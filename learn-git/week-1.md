## 2. 1주차 주요 개념 키워드
1. 커밋 (Commit)
    - Git은 commit(커밋)을 통해 현재 프로젝트의 상태를 저장하고 조회합니다.

2. 트래킹 (Tracking)
    - 로컬 레포지토리와 원격 레포지토리를 연결

3. 푸쉬 (Push)
    - 로컬 레포지토리의 커밋들을 원격 레포지토리 커밋들에 합치기

4. 풀 (Pull)
    - 원격 레포지토리의 커밋들은 로컬 레포지토리 커밋들에 합치기

5. 클론 (Clone)
    - 원격 레포지토리를 내 컴퓨터에 가져오기
